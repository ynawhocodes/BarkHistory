# BarkHistory
🗣 a.k.a 흑흑역사
## 서비스 소개
> **너의 흑역사(darkhistory)를 짖어라(bark)**  
개인마다 가지고 있는 흑역사를 익명으로 타인과 자유롭게 공유할 수 있는 커뮤니티
//## Overview

## Steps to run
```
1. git clone https://github.com/ynawhocodes/BarkHistory.git
2. cd BarkHistory
3. node main.js
```
